11. yeyen menuangkan bensin kedalam gelas ukur sebanyak 200 ml.jika massa jenis bensin 0,7 g/cm³,berapakah massa bensin yang dituangkan yeyen


diketahui:
V = 200 mL = 200 cm³
P = 0,7 gram/cm³

ditanya: massa = ?

Jawab:

P adalah massa jenis yang artinya besaran massa tiap satuan volumenya maka rumusnya dapat ditulis:
P = m/V
0,7 gram/cm³ = m/ 200 cm³
massa = 0,7 gram/cm³ x 200 cm³
massa = 140 gram

Jadi, massa bensin yang dituangkan Yeyen adalah 140 gram.

Waktu yang dibutuhkan Arik agar jarak motornya berada 35 km didepan motor Yeyen adalah 5 jam​

Pembahasan :

kecepatan adalah besarnya jarak yang ditempuh dalam setiap satuan waktu. Kecepatan salah satu besaran vektor yaitu besaran yang memiliki arah. Kecepatan ditentukan dari berpindahnya suatu benda berdasarkan banyaknya waktu yang dibutuhkan dalam berpindah. Kecepatan dibedakan menjadi kecepatan tetap dan rata-rata. Kecepatan tetap, adalah tetapan besarnya nilai kecepatan yang di ukur pada saat itu sedangkan kecepatan rata-rata adalah kecepatan yang di ambil dari banyaknya perubahan kecepatan.

Contoh dari satuan kecepatan, yaitu : km/jam, m/menit, m/detik, dan lain sebagainya. Rumus kecepatan adalah jarak dibagi waktu yang dapat ditulis sebagai berikut :

[tex]\boxed{\boxed{v = \frac{s}{t} }}[/tex]

[tex]\boxed{\boxed{t = \frac{s}{v} }}\\\\\boxed{\boxed{s = v\times t }}[/tex]

v = kecepatan

s = jarak

t = waktu

Berdasarkan rumus di atas akan digunakan untuk menghitung waktu tempuh dengan cara sebagai berikut :

Diketahui :

Va = 49 Km/jam

Vb = 42 Km/jam

s = 35 Km

Ditanyakan :

t pada kecepatan Vc

Jawab :

[tex]Vc = Va-Vb\\\\= 49 - 42\\\\= 7 Km/jam\\\\t =\frac{s}{v} \\\\= 35 : 7 \\\\= 5\ jam[/tex]

Jadi berdasarkan hitungan tersebut waktu tempuh yang dibutuhkan Arik agar jarak motornya berada 35 km didepan motor Yeyen adalah 5 jam. Jadi jawaban yang tepat itu yang D

38. 1. Buatlah program dengan memanfaatkan konsep tumpukan(stack) untuk permasalahan pemutaran playlist lagu dengan menggunakan Array. import java.util.Scanner;

public class Playlist {

   private static final int MAX_SIZE = 100;

   private String[] songs = new String[MAX_SIZE];

   private int top = -1;

   public static void main(String[] args) {

       Scanner input = new Scanner(System.in);

       Playlist playlist = new Playlist();

       int choice;

                   System.out.println("Lagu " + removedSong + " telah dihapus dari playlist");

               case 4:

                   System.out.print("Masukkan posisi playlist yang diinginkan: ");

                   System.out.print("Masukkan judul lagu: ");

                   playlist.pushAt(songTitle, index);

   public void push(String songTitle) {

       if (top == MAX_SIZE - 1) {

           System.out.println("Playlist penuh, tidak dapat menambahkan lagu");

       } else {

           top++;

           songs[top] = songTitle;

       }

   }

   public String pop() {

       if (top == -1) {

           System.out.println("Playlist kosong, tidak dapat menghapus lagu");

           return null;

       } else {

           String removedSong = songs[top];

           return removedSong;

       }

   public void pushAt(String songTitle, int index) {

       if (top == MAX_SIZE - 1) {

           System.out.println("Playlist penuh, tidak dapat menambahkan lagu");

       } else if (index < 0 || index > top + 1) {

           System.out.println("Posisi playlist tidak valid");

     

} else {

       for (int i = top; i >= index; i--) {

           songs[i + 1] = songs[i];

       }

       songs[index] = songTitle;

       top++;

   }

   public String popAt(int index) {

       if (top == -1) {

           System.out.println("Playlist kosong, tidak dapat menghapus lagu");

           return null;

       } else if (index < 0 || index > top) {

           System.out.println("Posisi playlist tidak valid");

           return null;

           String removedSong = songs[index];

           for (int i = index; i < top; i++) {

               songs[i] = songs[i + 1];

           }

           top--;

           return removedSong;

       }

   }

           }

       }

   }
